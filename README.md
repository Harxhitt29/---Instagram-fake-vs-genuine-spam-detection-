# 📸 Instagram Fake vs Genuine Spam Detection

## 📌 Description  
This project develops a machine learning system to classify Instagram accounts as fake (spam) or genuine (real). It analyzes features like followers, following ratio, posts, engagement, and profile details to detect suspicious behavior and improve social media security.

## 🚀 Features  
- Detect fake vs genuine Instagram accounts  
- Machine learning-based classification  
- Data preprocessing and feature extraction  
- Model evaluation using accuracy, precision, recall, F1-score  
- Easy-to-use prediction system
